How to Prompt SAM 3

1. Natural Language & Attributes

SAM 3 understands detailed descriptions. Use color, shape, and context to be specific.

Good Examples
  • "red car"
  • "person wearing white shirt"
  • "wooden chair next to the table"
Too Vague
  • "thing"
  • "object"
  • "stuff"

2. Spatial & Parts

You can target specific parts of an object or locations.

"wheel of the car" "handle of the mug" "left shoe"

3. Point Selection

Enable the "Click to Select Object" toggle in the configuration panel. Then, simply click on any object in the video feed to instantly segment it without typing a prompt.

Best Practices

  • Use high-resolution media (1080p+) for better detail.
  • Ensure good lighting and clear contrast.
  • Avoid heavy motion blur in videos.

Configuration Guide

Confidence Threshold

Determines how sure the AI needs to be before accepting a detection.

  • High (>70%): Fewer detections, but very accurate.
  • Low (<30%): More detections, but may include background noise.
  • Tip: If you see "Done" status but no masks, try lowering this to 20-30%.

Mask Threshold

Controls the "strictness" of the object boundaries.

  • High: Tighter masks, might miss thin parts (limbs, poles).
  • Low: Fuller masks, might spill over slightly.

Video Performance

Processing Interval

Skip frames to speed up processing.
• Low (1-5): Smooth, high GPU usage.
• High (10+): Fast, slightly choppy mask.

Input Resolution

Resize video before AI processing.
• 1024px: High detail, slower.
• 512px: Max speed, less detail.

Start with defaults (50%). If the object is complex or lighting is poor, lower the Confidence first.
